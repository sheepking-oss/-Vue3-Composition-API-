# -Vue3-Composition-API-
1.  首页：推荐歌单展示（卡片式布局，带封面、歌单名称、播放量）、热门歌曲排行榜、banner轮播图，风格贴合主流音乐APP，视觉简洁美观； 2.  播放器核心：底部固定播放栏（显示当前播放歌曲封面、歌名、歌手），支持播放/暂停、上一曲/下一曲、进度条拖拽、音量调节，支持单曲循环、列表循环、随机播放三种模式切换； 3.  歌单详情页：点击歌单可进入详情，展示歌单内所有歌曲列表（带序号、歌名、歌手、时长），支持点击歌曲直接播放，可勾选歌曲加入播放队列； 4.  个人中心简易版：展示个人收藏歌单、最近播放歌曲，支持删除最近播放记录； 5.  附加交互：播放歌曲时，封面有轻微旋转动画，切换歌曲时有过渡效果，hover歌单/歌曲时有悬浮动画，适配电脑端主流浏览器，响应式布局（适配平板、电脑）。
